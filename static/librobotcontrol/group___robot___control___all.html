Robot Control Library
All Modules

Single header that includes all modules.

<robotcontrol.h>

This includes every single module in the Robot Control API so that users do not need to include each module individually.

Use these modules by linking to /usr/lib/librobotcontrol.so

/**
* <robotcontrol.h>

* @brief Single header that includes all modules

#else

#endif

#include <rc/spi.h>
#include <rc/start_stop.h>
#include <rc/time.h>
#include <rc/uart.h>
#include <rc/version.h>
#endif // ROBOTCONTROL_H
